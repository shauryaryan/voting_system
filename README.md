Fingerprint Voting System

 Overview

The Fingerprint Voting System is a secure and efficient voting application designed to utilize biometric authentication for user registration, authentication, and voting processes. This system ensures that each vote is uniquely tied to an individual using their fingerprint, reducing the risk of fraud and enhancing the integrity of the voting process.

Features

- User Registration**: Register users by capturing their fingerprints and storing them securely in the database.
- User Authentication**: Authenticate users through fingerprint verification to ensure secure access to the voting system.
- Voting: Allow authenticated users to cast their votes for their preferred candidates.
- Admin Panel: Admins can view the total votes and monitor the voting process.

Technologies Used

- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- Database: MongoDB (via MongoDB Atlas)
- Biometric Integration: PyFingerprint for fingerprint capture and verification
